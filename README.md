# Danish  Trainer
This is a small experimental app to review danish words, verbs and numbers. 

Disclaimer: this a toy project, with lots of vibe coding, be warned.

## App
The app itself has been implemented with a heavy reliance on Codex as I am not an expert of HTML or JS.
I used pyscript for the logic as I wanted to try it out.
It is a static website for easier deployment.

## The data
Data is manually curated, as I am adding it slowly to the decks myself.
I add it under deckgen/danish/ and rely on deckgen.py to generate the final .json to be used as static assets for the app.
To create the audio recordings, I used a TTS model from huggingface.
